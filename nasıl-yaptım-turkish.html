BOOTSTRAP BİLGİLENDİRME

Bootstrap, HTML, CSS ve JavaScript ile yazılmış kullanışlı, yeniden kullanılabilir kod parçalarından oluşan dev bir koleksiyondur. Ayrıca, geliştiricilerin ve tasarımcıların hızla tam olarak duyarlı web siteleri oluşturmasını sağlayan bir frameworktür. Bootstrap web uygulaması geliştirme araç takımı, eski Twitter çalışanları Mark Otto ve Jacob Thornton tarafından oluşturulmuştur. Bootstrap ile geliştiriciler web uygulamaları geliştirirken, tasarım konusunda endişelenmeden geliştirme çalışmalarına odaklanabilir ve iyi görünümlü bir web sitesini hızlı bir şekilde çalışır hale getirebilir. Tersine, web tasarımcılarına ilginç Bootstrap temaları oluşturmak için sağlam bir temel sağlar.

KURULUM

Bootstrap Nasıl Kullanılır?

Bootstrap içerisinde hazır kodlar bulunduğu için kopyala yapıştır şeklinde tasarımlar yapabilirsiniz. Ancak tasarım yapabilmek için yerine getirmeniz gereken bazı işlemler var. Güncel bir tasarım aracı olduğu için sayfanız mutlaka HTML5 olmalıdır. HTML5 sayfanızı kullanıma hazır hale getirebilmek için jquery cript ve CSS dosyasını internet sayfanıza mutlaka eklemeniz gerekir. Bu işlemleri yaparak sayfanızdaki tüm tasarımların referans adresini getbootstrap.com olarak belirlersiniz. Siteye girdikten sonra ana sayfanın üst kısmında bulunan menüdeki Getting Started butonuna tıklayarak Bootstrap ile tasarım yapmaya başlayabilirsiniz.

  • Javascript, CSS ve Componet alanlarında sizi bekleyen kodları kopyaladıktan sonra yapıştırarak işlemlerinizi halledebilirsiniz. Bootstrap kütüphanesinin ana sayfası adresinden erişebilirsiniz.
    • Bu siteye geldiğimizde iki yöntem kullanabiliriz. Öncelikle siteye girdikten sonra önümüze gelen download tuşuna basıp tüm dosyaları indirdikten sonra işimize yarayanları kod dosyamıza taşıyabiliriz. Site açıldığında sizi bu sayfa karşılıyor

      Site açıldığında sizi bu sayfa karşılıyor burada download tuşuna bastığımızda ise yönlendirdiği sayfa bu şekilde;
      Burada ikinci tıkladığımız download tuşundan sonra otomatik rar uzantılı dosya inmeye başlayacak. Bu dosya indikten sonra kütüphanenizde size uygun olan çalışacağınız kütüphane uzantılarını kendiniz seçerek oluşturduğunuz kod dizinine dosyaları aktarmanız yeterlidir.
  • Eğer dosya indirmek istemiyorsanız ikinci yöntemi kullanabilirsiniz burası daha basit. Bootstrap sitesine geldiğinizde sayfanın alt kısmına indiğinizde jsDelivr kısmını göreceksiniz. Burada css ve Javascript linkleri hazır bulunuyor. Copy tuşunu basıp kopylayıp kod kısmıızda < head > etiketinin içine yazmanız yeterli. Kodlarınız bootstrap kütüphanesine entegre olmuş durumda bundan sonra kod yazmaya başlayabilirsiniz.
